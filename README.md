# Tanzawa: Tsurugi SQL console

Tsurugi SQL console (Tanzawa) is a text based SQL client program.

## Requirements

* JDK `>= 11`

## How to build

```sh
./gradlew assemble
```

## How to test

```sh
./gradle test
```

## How to run CLI

see [modules/cli/README.md](modules/cli/README.md).
